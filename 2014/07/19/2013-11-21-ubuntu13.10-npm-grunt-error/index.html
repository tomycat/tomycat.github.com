ubuntu 13.10 下面安装 grunt 报错 

$ npm install grunt #产生报错
npm ERR! Error: No compatible version found: underscore.string@'~2.2.0rc'
npm ERR! Valid install targets:

原因是ubuntu13.10中的npm版本过低

$ npm -v
1.2.18

解决办法 ，用npm重新安装npm

$ sudo npm install npm -g
